┌─────────────────────────────────────────────────────────────────────────────┐
│                           INPUT: "Java is good"                             │
└─────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                              1. TOKENIZATION                                │
│  "Java" → token_id(42)                                                      │
│  "is"   → token_id(15)                                                      │
│  "good" → token_id(87)                                                      │
└─────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                              2. EMBEDDING                                   │
│  Each token becomes a 64-dimension vector                                   │
│  X = [seq_len=3, embed_dim=64]                                              │
└─────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                        3. POSITIONAL ENCODING                               │
│  X = X + positional_encoding                                                │
└─────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌────────────────────────────────────────────────────────────────────────────┐
│                         4. FORWARD PASS (BLOCKS)                           │
│                                                                            │
│  ┌───────────────────────────────────────────────────────────────────────┐ │
│  │                      MULTI-HEAD ATTENTION                             │ │
│  │                                                                       │ │
│  │  Q = X × WQ    → [3, 64]                                              │ │
│  │  K = X × WK    → [3, 64]                                              │ │
│  │  V = X × WV    → [3, 64]                                              │ │
│  │                                                                       │ │
│  │  scores = Q × K^T           → [3, 3]                                  │ │
│  │  scores = scores / √64                                                │ │
│  │  weights = softmax(scores)   → [3, 3]                                 │ │
│  │                                                                       │ │
│  │  attention = weights × V     → [3, 64]                                │ │
│  │  attention = attention × WO  → [3, 64]                                │ │
│  │                                                                       │ │
│  │  X = X + attention  (residual)                                        │ │
│  │  X = layerNorm(X)                                                     │ │
│  └───────────────────────────────────────────────────────────────────────┘ │
│                                      │                                     │
│  ┌───────────────────────────────────────────────────────────────────────┐ │
│  │                      FEED-FORWARD NETWORK                             │ │
│  │                                                                       │ │
│  │  H = X × W1          → [3, 256]                                       │ │
│  │  H = ReLU(H)         → [3, 256] (zeros negative values)               │ │
│  │  ffn = H × W2        → [3, 64]                                        │ │
│  │                                                                       │ │
│  │  X = X + ffn  (residual)                                              │ │
│  │  X = layerNorm(X)                                                     │ │
│  └───────────────────────────────────────────────────────────────────────┘ │
│                                                                            │
│  Repeat for each block (2 blocks total)                                    │
└────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                         5. OUTPUT LAYER                                     │
│  logits = X × W_out  → [3, 5000]                                            │
└─────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                         6. LOSS CALCULATION                                 │
│  loss = cross_entropy(logits, target)                                       │
│  loss = -log(probs[target])  (for each position)                            │
└─────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌────────────────────────────────────────────────────────────────────────────┐
│                         7. BACKWARD PASS                                   │
│                                                                            │
│  ┌───────────────────────────────────────────────────────────────────────┐ │
│  │                      LOSS GRADIENT                                    │ │
│  │                                                                       │ │
│  │  grad_logits = dLoss/dLogits                                          │ │
│  │  grad_logits[i] = probs[i] - 1 (if i == target)                       │ │
│  │  grad_logits[i] = probs[i]     (if i != target)                       │ │
│  └───────────────────────────────────────────────────────────────────────┘ │
│                                      │                                     │
│  ┌───────────────────────────────────────────────────────────────────────┐ │
│  │                      OUTPUT LAYER BACKWARD                            │ │
│  │                                                                       │ │
│  │  grad_W_out = X^T × grad_logits                                       │ │
│  │  W_out = W_out - lr × grad_W_out                                      │ │
│  │                                                                       │ │
│  │  grad_X = grad_logits × W_out^T                                       │ │
│  └───────────────────────────────────────────────────────────────────────┘ │
│                                      │                                     │
│  ┌───────────────────────────────────────────────────────────────────────┐ │
│  │                      FFN BACKWARD (per block)                         │ │
│  │                                                                       │ │
│  │  grad_H = grad_X × W2^T                                               │ │
│  │  grad_H = grad_H (where H > 0, else 0)  # ReLU backward               │ │
│  │                                                                       │ │
│  │  grad_W2 = H^T × grad_X                                               │ │
│  │  W2 = W2 - lr × grad_W2                                               │ │
│  │                                                                       │ │
│  │  grad_W1 = X^T × grad_H                                               │ │
│  │  W1 = W1 - lr × grad_W1                                               │ │
│  │                                                                       │ │
│  │  grad_X_ffn = grad_H × W1^T                                           │ │
│  └───────────────────────────────────────────────────────────────────────┘ │
│                                      │                                     │
│  ┌───────────────────────────────────────────────────────────────────────┐ │
│  │                    ATTENTION BACKWARD (per block)                     │ │
│  │                                                                       │ │
│  │  grad_attention = grad_X                                              │ │
│  │  grad_WO = attention_input^T × grad_attention                         │ │
│  │  WO = WO - lr × grad_WO                                               │ │
│  │                                                                       │ │
│  │  grad_V = weights^T × grad_attention                                  │ │
│  │  WV = WV - lr × (X^T × grad_V)                                        │ │
│  │                                                                       │ │
│  │  grad_weights = grad_attention × V^T                                  │ │
│  │  grad_scores = grad_weights × softmax_derivative                      │ │
│  │                                                                       │ │
│  │  grad_Q = grad_scores × K                                             │ │
│  │  grad_K = grad_scores^T × Q                                           │ │
│  │                                                                       │ │
│  │  WQ = WQ - lr × (X^T × grad_Q)                                        │ │
│  │  WK = WK - lr × (X^T × grad_K)                                        │ │
│  │                                                                       │ │
│  │  grad_X_attn = grad_Q × WQ^T + grad_K × WK^T + grad_V × WV^T          │ │
│  └───────────────────────────────────────────────────────────────────────┘ │
│                                      │                                     │
│  ┌───────────────────────────────────────────────────────────────────────┐ │
│  │                    EMBEDDING BACKWARD                                 │ │
│  │                                                                       │ │
│  │  grad_embedding = grad_X_total                                        │ │
│  │  embedding_weights[id] -= lr × grad_embedding[i]                      │ │
│  └───────────────────────────────────────────────────────────────────────┘ │
└────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                         8. WEIGHT UPDATE                                    │
│  All weights (WQ, WK, WV, WO, W1, W2, W_out, embedding)                     │
│  are updated with gradient descent:                                         │
│                                                                             │
│  W = W - learning_rate × grad_W                                             │
└─────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                         9. NEXT EPOCH                                       │
│  Repeat entire process with the next batch of data                          │

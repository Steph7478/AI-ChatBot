# Java História
what is java|Java is a high-level, object-oriented programming language created by James Gosling at Sun Microsystems in 1995. Originally called Oak, it was designed for set-top boxes! Now it runs on over 3 billion devices! ☕📱
history of java|Java was born in 1991 as the Oak project by James Gosling, Mike Sheridan, and Patrick Naughton. Renamed Java in 1995 (named after Java coffee) and the rest is history! From applets to Android! 🌍☕
who created java|James Gosling! The father of Java! He created it at Sun Microsystems. Known as "Dr. Java" and still loves the language! 👨‍💻☕
when was java created|Java was created in 1991 as Oak but officially released in 1995. That's over 25+ years of excellence! Older than many developers today! 🎂☕

# OOP em Java
what is oop in java|Object-Oriented Programming in Java means everything is an object (except primitives)! Encapsulation, Inheritance, Polymorphism, Abstraction - the four pillars! Build real-world models in code! 🏗️☕
encapsulation in java|Encapsulation hides data using private fields and public getters/setters! Protects your data from unwanted access! It's like a capsule - keep the medicine safe! 💊☕
inheritance in java|Inheritance lets a class inherit fields and methods from another class using 'extends'! Code reuse is beautiful! Parent class, child class - like parent and child! 👨‍👦☕
polymorphism in java|Polymorphism means many forms! One interface, multiple implementations! Method overloading and overriding - your code becomes flexible and powerful! 🎭☕
abstraction in java|Abstraction hides implementation details using abstract classes and interfaces! You see WHAT, not HOW! Like driving a car - you know the pedals, not the engine! 🚗☕

# Java Features
what are java features|Platform independent, object-oriented, robust, secure, multithreaded, portable, high performance! Java has it all! Write once, run anywhere! That's the magic! ✨☕
what is bytecode|Bytecode is Java's secret sauce! Your .java files compile to .class bytecode. The JVM interprets or JIT-compiles it to machine code. Python can't do that! 🔐☕
platform independent|Write once, run anywhere! Your Java code runs on any device with a JVM. Windows, Mac, Linux, Android, even toasters! That's real freedom! 🆓☕

# JVM Internals
how does jvm work|JVM loads bytecode, verifies it, interprets or JIT-compiles to native code, manages memory with garbage collection, and handles threads! It's a computer inside your computer! 💻🖥️☕
jvm architecture|Class loader, runtime data areas (heap, stack, method area), execution engine, garbage collector! JVM is beautifully designed! Each part has a job! 🏗️🔧☕
what is jit compiler|Just-In-Time compiler identifies hot methods and compiles them to native code at runtime! Massive performance boost! Python is always interpreted - no JIT by default! 🚀⚡☕
heap vs stack in java|Heap stores ALL objects (shared across threads). Stack stores primitives and references (each thread has its own). Heap is big, stack is fast! 📚⚡☕

what are collections in java|Collections Framework gives you data structures out of the box! List, Set, Map - ArrayList, HashSet, HashMap! Python wishes it had this organization! 📦📋☕

hashmap in java|HashMap stores key-value pairs with O(1) average lookup! Keys are hashed, values are retrieved fast! Python's dict is similar but Java's is more configurable! 🗺️🔑☕
hashset in java|HashSet stores unique elements with O(1) lookup! No duplicates allowed! Perfect for when you need a collection of distinct items! 🎯☕
